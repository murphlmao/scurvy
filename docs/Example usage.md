# Example usage for each command

## Network
```sh
$ ip
# 192.168.50.11
```

```sh
$ mac
# 4b:48:d4:d0:90:69
```


## File System
```sh
# Directory Tree
$ dt . -e "incremental,deps,build,.fingerprint" 1
# params:
# 1 path
# 2 -e {exclude} "dir1, dir2, dir3..."
# depth

# output:
├─ .cargo-lock
├─ argh.d
├─ argh.exe
├─ argh.pdb
├─ dt.d
├─ dt.exe
├─ dt.pdb
├─ examples
├─ ip.d
├─ ip.exe
├─ ip.pdb
├─ libscurvy.d
├─ libscurvy.rlib
├─ mac.d
├─ mac.exe
├─ mac.pdb
├─ scurvy.d
├─ scurvy.exe
└─ scurvy.pdb
```
